Python3学习笔记-4

发表于 2022-04-10 更新于 2023-04-01 分类于 Python3

模块

os.path

系统模块中的路径模块

os.path.abspath()

该方法返回绝对路径，可以将相对路径转化为绝对路径

os.path.basename()

该方法返回路径中最后一个路径分隔符后的内容

os.path.dirname()

返回文件路径，即路径最后一个分隔符前的所有内容

os.path.join()

组合两个路径

os.path.split()

将路径分割为 dirname 和 basename，返回一个元组

os.path.splitdrive()

一般用在 Windows 下，返回驱动器名和路径组成的元组

os.path.splitext()

分割路径中的文件与扩展名

os.path.getsize()

返回文件大小，单位为 B，如果文件不存在就返回错误

os.path.isabs(), os.path.isfile(), os.path.isdir()

os.path.isabs() 判断是否为绝对路径，os.path.isfile() 判断是否为一个存在的文件，os.path.isdir() 判断是否为一个存在的目录

os.path.exists()

路径存在则返回 True，路径损坏返回 False

os.path.samefile()

判断目录或文件是否相同

os.path.getatime(), os.path.getmtime(), os.path.getctime()

os.path.getatime() 返回文件最近访问时间，os.path.getmtime() 返回文件最近的修改时间，os.path.getctime() 返回文件的创建时间

shutil

shutil 模块提供了一系列对文件和文件集合的高阶操作

shutil.copy()

将文件拷贝到其他文件或目录。copy() 会拷贝文件数据和文件的权限模式，其他元数据，例如文件的创建和修改时间不会被保留。要保留所有原有的元数据，请改用 copy2()

语法格式

shutil.copy(src, dst, *, follow_symlinks=True)

  • src：路径类对象字符串
  • dst：路径类对象字符串。如果 dst 指定了一个目录，文件将使用 src 中的基准文件名拷贝到 dst 中
  • follow_symlinks：如果 follow_symlinks 为假值且 src 为符号链接，则 dst 也将被创建为符号链接。如果 follow_symlinks 为真值且 src 为符号链接，则 dst 将成为 src 所指向的文件的一个副本

shutil.copy2()

类似于 copy()，区别在于 copy2() 还会尝试保留文件的元数据

语法格式

shutil.copy2(src, dst, *, follow_symlinks=True)

  • 参数说明同上

shutil.copyfile()

将文件内容（不包括元数据）拷贝到新的文件。源和目标必须代表一个文件，并且目标必须是可写的。如果目标已经存在，则将其替换为源文件，否则将创建一个新文件。如果源和目标表示相同的文件，则将引发 SameFileError 异常

语法格式

shutil.copyfile(src, dst, *, follow_symlinks=True)

  • 参数说明同上

shutil.copytree()

拷贝所有文件到新的文件夹下，保持原有的文件结构

语法格式

shutil.copytree(src, dst)

  • dst：指向的路径必须不存在

shutil.rmtree()

删除一个完整的目录树

语法格式

shutil.rmtree(path, ignore_errors=False, onerror=None)

  • path：必须指向一个目录
  • ignore_errors：如果为真，删除失败导致的错误将被忽略；如果为假，此类错误将通过调用由 onerror 所指定的处理程序来处理，或者如果此参数被省略则将引发一个异常
  • onerror：如果提供了 onerror，它必须为接受三个形参的可调用对象: function, path 和 excinfo
  • 第一个形参 function 是引发异常的函数；它依赖于具体的平台和实现。 第二个形参 path 将是传递给 function 的路径名。 第三个形参 excinfo 将是由 sys.exc_info() 所返回的异常信息。 由 onerror 所引发的异常将不会被捕获

shutil.move()

递归地将一个文件或目录移至另一个位置。如果目标是已存在的目录，则 src 会被移至该目录下。如果目标已存在但不是目录，它可能会被覆盖，具体取决于 os.rename() 的语义

语法格式

shutil.move(src, dst, copy_function=copy2)

  • src：源目录
  • dst：目标目录
  • copy_function：如果给出了 copy_function，则它必须为接受两个参数 src 和 dst 的可调用对象，并将在 os.rename() 无法使用时被用来将 src 拷贝到 dst。 如果源是一个目录，则会调用 copytree()，并向它传入 copy_function()。 默认的 copy_function 是 copy2()。 使用 copy() 作为 copy_function 允许在无法附带拷贝元数据时让移动操作成功执行，但其代价是不拷贝任何元数据

zipfile

该模块提供了创建、读取、写入、添加及列出 ZIP 文件的工具

class zipfile.ZipInfo

用于标识档案内一个成员信息的类。此类的实例会由 ZipFile 对象的 getinfo() 和 infolist() 方法返回。大多数 zipfile 模块的用户都不必创建它们，只需使用此模块所创建的实例

语法格式

class zipfile.ZipInfo(filename='NoName', date_time=1980, 1, 1, 0, 0, 0)

  • filename：档案成员的全名
  • date_time： 包含六个字段的描述最近修改时间的元组

class zipfile.ZipFile

用于读写 ZIP 文件的类

语法格式

class zipfile.ZipFile(file, mode='r', compression=ZIP_STORED, allowZip64=True, compresslevel=None, *, strict_timestamps=True)

  • file：一个指向文件的路径（字符串），一个类文件对象或者一个 path-like object
  • mode：r 读取一个存在的文件，w 截断并写入新的文件， a 添加到一个存在的文件，x 仅新建并写入新的文件。
  • compression：在写入归档时要使用的 ZIP 压缩方法，应为 ZIP_STORED, ZIP_DEFLATED, ZIP_BZIP2 或 ZIP_LZMA；不可识别值将导致引发 NotImplementedError
  • allowZip64：如果为 True，则当 zipfile 大于 4GiB 时将创建使用 ZIP64 扩展的 ZIP 文件。如果为 False， 则当 ZIP 文件需要 ZIP64 扩展时将引发异常
  • compresslevel：该形参用于控制在将文件写入归档时要使用的压缩等级。当使用 ZIP_STORED 或 ZIP_LZMA 时无压缩效果。当使用 ZIPDEFLATED 时接受整数 0 至 9。当使用 ZIP_BZIP2 时接受整数 1 至 9
  • strict_timestamps：为 False 时允许压缩早于 1980-01-01 的文件，代价是会将时间戳设为 1980-01-01。而 2107-12-31 为上限值

ZipFile.close()

关闭归档文件，必须在退出程序之前调用 close() 否则将不会写入关键记录数据。可用 with 语句替代

ZipFile.getinfo(name)

返回一个 ZipInfo 对象，其中包含有关归档成员 name 的信息

ZipFile.infolist()

返回一个列表，其中包含每个归档成员的 ZipInfo 对象。如果是打开一个现有归档则这些对象的排列顺序与它们对应条目在磁盘上的实际 ZIP 文件中的顺序一致

ZipFile.namelist()

返回按名称排序的归档成员的列表

ZipFile.open()

以二进制文件类对象的形式访问一个归档成员

语法格式

ZipFile.open(name, mode='r', pwd=None, *, force_zip64=False)

  • name：可以是归档内某个文件的名称也可以是某个 ZipInfo 对象
  • mode：必须为 r 或 w
  • pwd：用于解密已加密 ZIP 文件的密码
  • force_zip64：当写入一个文件时，如果文件大小不能预先确定但是可能超过 2 GiB，可传入 force_zip64=True 以确保标头格式能够支持超大文件

ZipFile.write()

将文件写入归档，归档必须使用 w，x 或 a 模式打开

语法格式

ZipFile.write(filename, arcname=None, compress_type=None, compresslevel=None)

  • filename：需要被写入归档的文件名
  • arcname：给予的归档名，默认情况下将与 filename 一致，但是不带驱动器盘符并会移除开头的路径分隔符
  • compress_type：如果给出，它将覆盖作为构造器 compression 形参对于新条目所给出的值
  • compresslevel：如果给出，也将覆盖构造器

ZipFile.extract()

从归档中提取出一个成员放入当前工作目录

语法格式

ZipFile.extract(member, path=None, pwd=None)

  • member：必须为成员的完整名或 ZipInfo 对象
  • path：指定一个要提取到的不同目录
  • pwd：用于解密文件的密码

ZipFile.extractall()

从归档中提取出所有成员放入当前工作目录

语法格式

ZipFile.extractall(path=None, members=None, pwd=None)

  • path：指定一个要提取到的不同目录
  • members：可选项，必须为 namelist() 所返回列表的一个子集
  • pwd：用于解密文件的密码

time

时间模块，Python 函数常用一个元组装起来的 9 组数字来处理时间

序号 字段 属性 值
0 4位数年 tm_year 2008
1 月 tm_mon 1 到 12
2 日 tm_mday 1 到 31
3 小时 tm_hour 0 到 23
4 分钟 tm_min 0 到 59
5 秒 tm_sec 0 到 61（60 或 61 是闰秒）
6 一周的第几日 tm_wday 0 到 6（0 是周一）
7 一年的第几日 tm_yday 1 到 366 （儒略历）
8 夏令时 tm_isdst -1, 0, 1, -1 是决定是否为夏令时的旗帜

time.time()

用于获取当前时间戳，时间戳单位最适于做日期运算。但 1970 年之前的日期就无法以此表示了。太遥远的日期也不行，UNIX 和 Windows 只支持到 2038 年

time.localtime()

从返回浮点数的时间戳方式向时间元组转换，只需将浮点数传递给 localtime() 之类的函数

time.asctime()

转换由 gmtime() 或 localtime() 所返回的表示时间的元组或 struct_time 为以下形式的字符串 Sun Jun 20 23:21:05 1993

time.strftime()

转换一个元组或 struct_time 表示的由 gmtime() 或 localtime() 返回的时间到由 format 参数指定的字符串

Python3 中时间日期格式化符号

  • %y 两位数的年份表示（00-99）
  • %Y 四位数的年份表示（000-9999）
  • %m 月份（01-12）
  • %d 月内中的一天（0-31）
  • %H 24小时制小时数（0-23）
  • %I 12小时制小时数（01-12）
  • %M 分钟数（00=59）
  • %S 秒（00-59）
  • %a 本地简化星期名称
  • %A 本地完整星期名称
  • %b 本地简化的月份名称
  • %B 本地完整的月份名称
  • %c 本地相应的日期表示和时间表示
  • %j 年内的一天（001-366）
  • %p 本地A.M.或P.M.的等价符
  • %U 一年中的星期数（00-53）星期天为星期的开始
  • %w 星期（0-6），星期天为星期的开始
  • %W 一年中的星期数（00-53）星期一为星期的开始
  • %x 本地相应的日期表示
  • %X 本地相应的时间表示
  • %Z 当前时区的名称
  • %% %号本身

time.sleep()

调用该方法的线程将被暂停执行，单位是秒。参数可以是浮点数，以表示更精确的睡眠时长

calendar

该模块可以用来输出日历，并提供日历相关的实用函数。默认情况下，这些日历把星期一当作一周的第一天，星期天为一周的最后一天，更改设置需要调用 calendar.setfirstweekday() 函数

class calendar.TextCalendar()

可以使用这个类生成纯文本日历。TextCalendar 实例有以下方法

formatmonth()

返回一个多行字符串来表示指定年月的日历

语法格式

formatmonth(theyear, themonth, w=0, l=0)

  • theyear：指定的年
  • themonth：指定的月份
  • w：日期的宽度，但始终保持日期居中
  • l：指定了每星期占用的行数
promonth()

与 formatmonth() 方法一样，返回一个月的日历。但是该方法有返回值，且不需要 print() 即可输出

语法格式

prmonth(theyear, themonth, w=0, l=0)

fromatyear()

返回一个多行字符串，这个字符串为一个 m 列日历

语法格式

formatyear(theyear, w=2, l=1, c=6, m=3)

  • theyear：需要指定的年份
  • w：日期列数
  • l：周的行数
  • c：月之间的间隔
  • m：一行的月份列数
pryear()

与 formatyear() 方法一样，返回一整年的日历。但是该方法有返回值且不需要 print() 即可输出

语法格式

pryear(theyear, w=2, l=1, c=6, m=3)

  • theyear：需要指定的年份
  • w：日期列数
  • l：周的行数
  • c：月之间的间隔
  • m：一行的月份列数

calendar.calendar()

使用 TextCalendar 类的 formatmonth() 以多行字符串形式返回月份日历

语法格式

calendar.calendar(year, w, l, c)

  • year：指定的年份
  • w：日期列数
  • l：周的行数
  • c：月之间的间隔

calendar.month()

使用 TextCalendar 类的 formatmonth() 以多行字符串形式返回月份日历

语法格式

calendar.month(theyear, themonth, w=0, l=0)

  • theyear：指定的年
  • themonth：指定的月份
  • w：日期的宽度，但始终保持日期居中
  • l：指定了每星期占用的行数

operator

operator 模块提供了一套与 Python 的内置运算符对应的高效率函数。例如，operator.add(x,y) 与 x+y 相同。

函数包含的种类有：对象的比较运算、逻辑运算、数学运算以及序列运算

operator.lt(a, b)

等价于 a<b

operator.le(a, b)

等价于 a<=b

operator.eq(a, b)

等价于 a==b

operator.ne(a, b)

等价于 a!=b

operator.gt(a, b)

等价于 a>b

operator.ge(a, b)

等价于 a>=b

operator.not(obj)

返回 not obj

operator.truth(obj)

obj 为真返回 True，否则返回 False

operator.is_(a, b)

返回 a is b，检测对象标识

operator.is_not(a, b)

返回 a is not b，检测对象标识

operator.abs(obj)

返回 obj 的绝对值

operator.add(a, b)

返回 a+b

operator.and_(a, b)

返回 a，b 按位与的结果

operator.floordiv(a, b)

返回 a//b

operator.index(a)

返回 a 转换为整数（int）的结果

operator.inv(obj), operator.invert(obj)

返回 obj 按位取反的结果

operator.lshift(a, b)

返回 a 左移 b 位的结果

operator.rshift(a, b)

返回 a 右移 b 位的结果

operator.mod(a, b)

返回 a%b

operator.mul(a, b)

返回 a*b

operator.matmul(a, b)

返回 a@b，矩阵乘法

operator.neg(obj)

返回 obj 取负的结果

operator.or_(a, b)

返回 a, b 按位或的结果

operator.pos(obj)

返回 obj 取正的结果

operator.pow(a, b)

返回 a**b

operator.sub(a, b)

返回 a-b

operator.truediv(a, b)

返回 a/b

operator.xor(a, b)

返回 a，b按位异或的结果

operator.concat(a, b)

对于序列 a, b 返回 a+b

operator.contains(a, b)

返回 b in a 检测的结果，需要注意操作数是反序的

operator.countOf(a, b)

返回 b 在 a 中出现的次数

operator.delitem(a, b)

移除 a 中索引号为 b 的值

operator.getitem(a, b)

返回 a 中索引为 b 的值

operator.indexOf(a, b)

返回 b 在 a 中首次出现所在的索引号

operator.setitem(a, b, c)

将 a 中索引号为 b 的值设为 c

operator.length_hint(obj, default=0)

返回对象 obj 的估计长度。首先尝试返回其实际长度，再使用 object.__length_hint__() 得出估计值，最后返回默认值
