我的ARK单人游戏设置

发表于 2022-01-15 更新于 2023-04-01 分类于 Games

游戏设置

阻止全局时停

  • Steam -> ARK: Survival Ecolved -> 属性 -> 通用 -> 启动选项 -> 添加代码 -preventhibernation

中文补丁替换

  • 游戏原版的中文会导致一些 BUG，因此建议使用中文补丁替换原中文文件
  • 路径：ARK\ShooterGame\Content\Localization\Game\zh
  • 替换文件：ShooterGame.locres
  • 相关网址：<http://c.tieba.baidu.com/p/7881686866?pid=144457650554&cid=#144457650554>

锁帧率设置

  • 路径：ARK\ShooterGame\Config
  • 修改文件：DefaultEngine.ini
  • 修改项：在 [/Script/Engine.RendererSettings] 项下面新增一行 t.MaxFps=90
  • 参数说明：设置最大帧率为 90

不使用 MOD 修改物品叠加限制

  • 路径：ARK\ShooterGame\Saved\Config\WindowsNoEditor
  • 修改文件：GameUserSettings.ini
  • 修改项：ItemStackSizeMultiplier
  • 默认值：1.000000
  • 修改值：1.0

通过配置文件修改恐龙等级上限

  • 路径：ARK\ShooterGame\Saved\Config\WindowsNoEditor
  • 修改文件：GameUserSettings.ini
  • 修改项：在 [ServerSettings] 项下面新增一行 OverrideOfficialDifficulty=5.000000
  • 默认值：5.0
  • 修改值：7.0
  • 参数说明：默认值为 5.0 代表着游戏的默认最高难度。恐龙最高等级是参数的 30 倍，即 5 对应的恐龙最高等级为 150

清除野生恐龙代码

  • 进入游戏，呼出控制台，输入 cheat destroywilddinos

建筑物拾取时间修改

  • 路径：ARK\ShooterGame\Saved\Config\WindowsNoEditor
  • 修改文件：GameUserSettings.ini
  • 修改项：StructurePickupTimeAfterPlacement
  • 默认值：30.000000(s)
  • 修改值：120.0
  • 设置永久可拾取：添加一行 AlwaysAllowStructurePickup=True

设置一次留满痕

  • 路径：ARK\ShooterGame\Saved\Config\WindowsNoEditor
  • 修改文件：GameUserSettings.ini
  • 修改项
    1. 修改留痕间隔系数 0.01
    2. [/Script/ShooterGame.ShooterGameUserSettings] 下 新增 BabyImprintAmountMultiplier=10000

代码设置恐龙颜色

  • 在控制台使用代码 SetTargetDinoColor 0 1 对目标设置颜色，第一个数字代表色区，范围0-5。第二个数字代表颜色代号。

本地文件修改人物，恐龙加点上限

  • 路径：ARK\ShooterGame\Saved\Config\WindowsNoEditor
  • 修改文件：Game.ini
  • 修改项
    • 将相关配置复制粘贴至Game.ini最后即可

本地文件修改终端上传物品，恐龙数量上限

  • 路径：ARK\ShooterGame\Saved\Config\WindowsNoEditor
  • 修改文件：GameUserSettings.ini
  • 修改项
    • 物品：在 [ServerSettings] 下新增 MaxTributeItems=150
    • 恐龙：在 [ServerSettings] 下新增 MaxTributeDinos=270

Custom Dino Levels

该模组可以更加详细的设定野生恐龙的等级范围。有三种设定模式

  1. 仙境模式
    该模式下恐龙等级会同仙境、中心岛和水晶岛有相同的生成模式：尽管水平权重总体上仍向较低范围倾斜，但是高等级恐龙有了更高的生成概率。一些等级的恐龙将永远不会被生成：在最高难度下，这些等级是 5, 10, 30, 35, 40, 65, 70, 75, 80, 105, 110, 115, 120 和 125（该模组并不会主动将游戏设置为最高难度，如果需要，参考本文前面的“通过配置文件修改等级上限”）

  2. 均衡模式（默认模式）
    该模式允许生成全等级范围的恐龙，5 级与 150 级的恐龙生成的概率是相同的。如果你需要自定义最小最大等级，该模式是必须激活的模式

  3. 高等级模式
    该模式反转了原始游戏中高低等级恐龙生成的概率，因此将严重偏向生成高等级恐龙

配置

  • 路径：ARK\ShooterGame\Saved\Config\WindowsNoEditor
  • 修改文件：GameUserSettings.ini

  • 修改项：[CustomLevelDistrib]

    • 参数说明

      • WantsRagLevels, WantsEqualLevels, WantsHighLevels 三项中有且只有一项为 True
      • 恐龙等级由基础难度与设定范围相乘所得，如5*30=150，即恐龙等级为150
      • MinLevel：设定普通恐龙等级范围的最小值
      • MaxLevel：设定普通恐龙等级范围的最大值
      • PreventBee：阻止等级影响到蜂王的等级，即蜂王等级还是游戏原设定
      • PreventTitan：阻止泰坦受影响
      • MinDrakeLevel：设定飞龙，岩龙的最小等级范围
      • MaxDrakeLevel：设定飞龙，岩龙最大的等级范围

    • WantsRagLevels 类仙境恐龙等级设置样例

    • WantsHighLevels 高等级恐龙设定样例

    • WantsEqualLevels 平均恐龙等级设定样例

配置

ARK Additions: The Collection

该模组添加了一系列新的恐龙

Acrocanthosaurus 高脊龙

野生高脊龙是一种食肉类恐龙，体型比霸王龙更大，对越小的目标造成的伤害越高。当他们受到伤害时会将高大的脊背露出来抵挡伤害（盾牌姿态），同时如果受到伤害过多（35% 最大生命值），他们会进入愤怒状态，这种状态下可以造成更高的伤害

高脊龙对普通的麻醉方式免疫，可以尝试通过激怒高脊龙在其吼叫时直接将麻醉药品放入其口中的方式驯服

分布

高脊龙可能出现在孤岛、焦土、中心岛、灭绝、创世纪一、仙境、瓦尔盖罗、水晶岛、迷失岛地图中

掉落物

可以掉落兽皮、生肉、优质生肉以及高脊龙肾上腺

高脊龙肾上腺是创造野蛮高脊龙的贡品

对抗

想要杀死高脊龙，最好使用体型相近或更大的恐龙，这样可以保证受到的伤害最小。高脊龙在正常情况下跑得比生存者更快，因此，如果有必要可以通过造成一定伤害（最大生命值的 10%）来激活其盾牌姿态降低其移动速度

高脊龙在激活肾上腺素的情况下几乎可以造成两倍伤害

在对抗一只野生的或者没有骑手的高脊龙时，可以在其激活盾牌姿态后停止攻击，等待它从盾牌状态退出，这样就不会再激活肾上腺素状态

驯服

高脊龙的驯服食物依次为卓越饲料、生羊肉、熟羊肉、优质生肉、优质熟肉、优质生鱼肉、优质熟鱼肉、生肉等

在驯服高脊龙时，需要将麻醉剂放到物品栏最后一格，然后攻击高脊龙使其进入咆哮状态，然后按 E 将麻醉剂从其口中送入，直到高脊龙倒地

可以尝试使用飞龙来驯服高脊龙，在困住高脊龙后，使用飞龙（推荐冰龙）来对高脊龙造成伤害并使其吼叫。火焰飞龙、闪电飞龙、毒龙不太推荐，因为可能影响驯养有效性

控制
  • 左键普通撕咬，当激活肾上腺素时攻速变快
  • 右键激活盾牌姿态，在这种姿态下受到最大生命值的 10% 伤害可以进入肾上腺激活状态。在盾牌姿态下使用左键可以使用盾牌撞击，可以对金属建筑造成伤害
  • C 键进行践踏攻击，攻击伤害取决于被攻击对象与高脊龙的体重对比，在肾上腺状态下可以践踏 3 次
  • Ctrl 键吼叫
扮演角色
  • 坦克。高脊龙可以担当坦克角色，当激活盾牌状态时，受到伤害会积累肾上腺素值，一旦激活肾上腺状态，将会获得一个 35s 的 Buff：美妙 0.2% 生命值恢复，并且增加近战伤害，抗性和移动速度
  • 顶级掠食者。高脊龙在面对体型较小的恐龙时有压倒性优势，但面对同级别及更高级的恐龙，如霸王龙和南巨时会有些吃力，推荐加点生命和攻击
  • 吸收子弹。激活盾牌姿态吸收子弹
  • Boss 战斗。高脊龙可以被带入 Boss 竞技场参与击杀 Boss
  • 石头人猎杀者。高脊龙的盾牌攻击不会被石头人减伤
Summon Savage Acro 野蛮高脊龙 Boss

使用高脊龙肾上腺腺体等材料制作野蛮高脊龙 Boss 的召唤物品。如果召唤就没有回头路，因此建议在空旷的场地召唤野蛮高脊龙。他的强度与君王泰坦相当。

Brachiosaurus 腕龙

腕龙是一种温顺的食草性恐龙，当受到伤害时，他们会采用后脚站地的方式来保护自己，同时会将全身的重量用于践踏攻击。驯服后的腕龙可以采集木头、茅草、石头、金属等物品

分布

孤岛、中心岛、仙境、畸变、瓦尔盖罗、创世纪一和水晶岛

掉落物

兽皮、生肉、优质生肉

对抗

应当使用灵活的恐龙或飞龙来对付腕龙。也可以使用步枪风筝腕龙，或使用模组生物高脊龙激活肾上腺状态来击杀腕龙

腕龙的吼叫可能导致任何生物的恐惧逃跑，因此在面对敌对腕龙时要特别注意它的吼叫

使用流血效果的武器或恐龙可以更好的击杀腕龙

驯服

腕龙的驯服食物依次为卓越饲料、蔬菜、紫色浆果以及其他浆果。腕龙的眩晕值恢复的很快，因此请确保有足够的麻醉药使其保持眩晕状态

腕龙无法被正常的方式麻醉，需要在其抬高前腿时攻击后腿前部，当受到足够伤害时，腕龙就会眩晕。建议使用远程武器风筝

控制
  • 左键使用一条前腿进行践踏
  • 右键使用一条后腿进行践踏
  • C 键触发双腿站立，这时使用左键进行双腿践踏
  • Ctrl 键发出吼叫，可以恐惧大部分生物
  • R 键循环采集效率，可以更改不同物资的采集效率
扮演角色
  • 采集者。腕龙可以采集木头、石头、金属、浆果、茅草等物资。对木头、石头、金属有 50% 减重效果
  • 攻城武器。腕龙可以对所有类型的建筑造成大量伤害
  • 毁灭者。腕龙的双脚践踏有很高的伤害，但是有 30s 的冷却时间，因此要在合适的时机释放
  • 恐吓者。腕龙的吼叫可以让大部分生物逃跑
  • 造粪机器。腕龙可以产生大量粪便
  • 驯服助手。腕龙可以吼叫使得其他恐龙恐惧，被恐惧的恐龙将降低抗性，更容易被击晕。另外如果野生恐龙在被恐惧的情况下被击晕驯服，那么它起身时就会攻击让它恐惧的腕龙，通过收回冷冻仓再放出来可以解决这个问题
  • 嘲讽者。有些生物在听到腕龙的吼叫后不会逃跑反而会攻击腕龙。同时也有一些生物完全不受腕龙吼叫影响

Concavenator 驼背龙

驼背龙通常出现在沙地里，2-3 只一组，潜游在沙土之中。驼背龙有一个背包加成，这使得他们可以扬沙使得敌人窒息。驼背龙通常是一击就跑的典型

分布

焦土、仙境、畸变、灭绝、创世纪一、水晶岛

掉落物

兽皮、角质、生肉、优质生肉

对抗

当遇到幸存者时，驼背龙会快速潜入地下不断攻击幸存者，直到他们被炸药逼出地底

推荐使用远程武器对抗驼背龙

没有氧气条的恐龙不受扬尘的影响，带有潜水装备或防毒面具的幸存者也不受扬尘影响，否则可能会被窒息（恐龙则是其他属性下降）

驯服

驼背龙的驯服食物为上等饲料、生羊肉、熟羊肉、优质生肉等

要驯服驼背龙，首先需要使用 C4 将他们炸出地面，然后使用正常方式麻醉他们。虽然驼背龙可以被正常方式麻醉，但是推荐使用炸药的方式使其眩晕，因为他们脱离地底的时间很短，同时他们的眩晕值下降很快，使用麻醉镖等方式可能在眩晕前就将它们击杀。可以使用一堆炸药然后引爆的方式增加大量眩晕值，然后在快眩晕时只引爆一个炸药避免有效性降低

推荐在驯服时骑乘一个移速较快的恐龙，如似鸡龙或剑齿虎

也可以尝试使用冰龙来驯服

控制
  • 左键使用嘴咬
  • 右键切换姿态，进入潜泳模式
  • C 键扬沙
  • 空格键在潜泳状态下跳跃
扮演角色
  • 地底旅行者。可以骑乘他们在沙地快速移动
  • 群体控制。扬沙可以造成范围伤害与控制

Cryolophosaurus 冰脊龙

冰脊龙体型娇小，他们的喉咙里有一种冷冻剂，通过吸收水或雪可以填充。冰脊龙可以将冷冻剂吐出来形成一团有麻痹效果的气体，也可以把它呼出来形成一股寒冷的气流。如果它成功冻结一个个体，冰脊龙将进入一个兴奋状态，提高攻速、伤害和抗性。冰脊龙对于传统的驯服方式免疫，但是当他们呼出气体时他们的喉咙十分脆弱，这时可以向他们嘴里射击来使得他们冻结自己，从而被喂食

分布

孤岛、中心岛、仙境、瓦尔盖罗、水晶岛

掉落物

兽皮、生肉

对抗

利用捕兽夹来阻止其移动，然后使用武器攻击。注意使用远程武器，并且需要规避他们的冰冻吐息和麻痹吐息

因为冰脊龙的生命值并不高，所以可以使用高等级高生物权限的恐龙直接击杀冰脊龙

驯服

冰脊龙驯服食物为卓越饲料、生羊肉等

为了驯服冰脊龙，你需要在它喷射冰霜吐息时向他的头部射击，这会导致其低温吐息将自身冻住，然后就可以被喂养食物

控制
  • 左键普通攻击。当进入瞄准状态时，按下左键将喷射冰冻吐息
  • 右键可以让冰脊龙通过水或雪填充冷冻液
  • C 键发射麻痹吐息
  • Ctrl 改变攻击姿态，切换为瞄准模式
扮演角色
  • 铠甲匠。当冰脊龙被驯服后，可以通过冰冻吐息来为友军增加一层 70% 减伤的冰霜护甲，但是会极大地降低他们的移动速度
  • 驯服助手。他们的冷冻吐息和麻醉吐息可以很方便的帮助驯服其他恐龙。冰脊龙的冷冻液值取决于氧气值，麻醉效果取决于攻击力
