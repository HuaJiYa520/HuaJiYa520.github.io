阿里云CentOS搭建饥荒服务器

发表于 2022-08-04 更新于 2023-04-01 分类于 Games

配置信息

操作系统：CentOS 8.5
饥荒版本：v518018

服务器搭建

命令格式

  • [local] 符号表示任意用户均可执行该条命令
  • [root] 符号表示需要使用 root 用户执行命令，如果是一般用户，执行命令时需要使用 sudo 前缀
  • (output) 表示命令输出结果

搭建步骤

  1. 可以考虑创建一个专门的 dst(Don’t Starve Together) 的用户,也可以直接使用 root 用户

  2. 安装相关依赖

  3. 安装 steamCMD, 并下载饥荒的服务器文件

  4. 检查是否缺乏相关依赖

  5. 如果有报错，可参考以下内容

  6. 试运行服务器

    这一步会在 ~/.klei/DoNotStarveTogether/ 下生成一个 MyDediServer 文件夹，其中 Master 指地上，Caves 指洞穴

  7. 可删除 MyDediServer 文件夹 rm -f ~/.klei/DoNotStarveTogether/MyDediServer，因为该文件夹后续将会被替换

  8. 配置 token
    1. 打开游戏客户端，找到 Account 按钮，点击进入用户界面
    2. 在上方的菜单栏中选择 游戏，再选择 《饥荒：联机版》的游戏服务器
    3. 如果没有建过服务器，则选择 添加新服务器
    4. 选择配置服务器
    5. 配置服务器为想要的设置，然后下载设置
    6. 下载后的文件为 MyDediServer，其中包含文件 Master, Caves, cluster.ini, cluster_token.ini
  9. 在本地新建一个存档，配置好想要的世界设置以及模组。加载好服务器到选人界面退出即可
  10. 退出后在 Steam 库存中右键饥荒，打开它的本地文件，找到刚刚创建的服务器存档，如果是第一个档，名字就是 Cluster_1，如果是第二个就是 Cluster_2，以此类推。也可以直接在游戏中选择对应存档，点击右上角扳手图标，选择“打开本地文件”
  11. 将 Cluster_1 中 Master 和 Caves 中的除 server.ini 外的文件复制到刚刚下载的 MyDediServer 对应的文件夹中
  12. 将 MyDediServer 复制到 DoNotStarveTogether 目录下
    1. 在当前文件夹打开 cmd 或 git bash 使用 scp -r MyDediServer root@xxx:~/.klei/DoNotStarveTogether/ 命令上传
    2. 或使用 ftp 工具上传

  13. 如果有配置 mod，则需要修改 ~/dst/mods/dedicated_server_mods_setup.lua 文件，将需要加载的 mod id 添加进该文件。可以通过 MyDediServer/Master/modoverrides.lua 文件中的 ["workshop-XXXXXXXXX"] 中得到 mod id 或直接去创意工坊查看对应 mod id

  14. 启动服务器

    1. 方式一：使用一个脚本同时启动地上与洞穴。在 /root 下新建脚本 startup.sh

      赋予脚本执行权限

      开启一个虚拟窗口并运行脚本

      成功启动后按下 Ctrl+A, D 最小化虚拟窗口，使其在后台运行

    2. 方式二：分别运行地上与地下的启动脚本

服务器世界设定

主世界

  • 世界选项

    • 冒险家
      • 额外起始资源：总是

  • 世界生成

    • 巨石：较多
    • 燧石：较多

洞穴

保持默认

模组

可以打开 选项——高级——自动订阅模组

客户端 mod

客户端 Mod 不影响进入服务器，可根据需要选择下载。已下为我的 Mod 配置

已启用

  • ActionQueue RB2 (RWYS supported), id=2325441848。动作队列 Mod
  • 排队论加强, id=2753482847。在 ActionQueue 的基础上增加一些功能
  • Combined Status, id=376333686。增强的人物状态显示
  • Craft Pot [DS, ROG, SW, DST], id=727774324。在烹饪时显示配方等信息
  • [DST] Nightmare HUD, id=1992293314。暗黑系 UI 风格
  • Geometric Placement, id=351325790。几何种地必备
  • Minimap HUD, id=345692228。增加一个小地图
  • Show Nicknames 显示昵称（客户端）, id=956206484。在人物头上显示他们的昵称
  • Wormhole Icons (Fixed Again), id=2831613121。添加虫洞标记
  • Celestial HUD, id=2571443104。天空色 UI

未启用

  • [DST]HD Item Icon 高清图标 - Shang, id=2260439333。物品图标的高清化，但是新版还没有更新
