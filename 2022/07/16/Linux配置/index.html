更换默认终端为 oh-my-zsh

  1. 查看当前使用的 Shell 类型，一般为 /bin/bash 或 /usr/bin/bash
    echo $0

  2. 查看当前默认终端，一般为 /bin/bash
    echo $SHELL

    （上图已修改 bash 为 zsh）

  3. 查看当前系统存在的 shell
    cat /etc/shells

  4. 下载 zsh
    yum -y install zsh
    or
    apt -y install zsh

  5. 更改默认终端，使用以下三种方法之一即可

    1. 使用命令 chsh -shell /bin/zsh username 更改默认 Shell
      1. -shell 参数可替换为 -s
      2. /bin/zsh 也可以换成其他想要更改的 shell
      3. username 替换为用户名，如 root
    2. 如果 chsh 命令不存在或执行失败，可以使用 usermod --shell /bin/zsh username，使用规则同上

    3. 手动修改 passwd 文件 vim /etc/passwd

      1. 文件第一行即为 root 用户使用的 shell
      2. username 一行即为 username 用户使用的 shell
      3. 更改 shell 路径为想要的路径即可

  6. 重启服务器，此时打开终端时应当为 oh-my-zsh

安装 oh-my-zsh

按照 oh-my-zsh 官方说明

  • 使用 curl 安装
    sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
  • 使用 wget 安装
    sh -c "$(wget https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh -O -)"

oh-my-zhs 更换主题

选用 Powerlevel10k 这个主题

官方链接为 <https://github.com/romkatv/powerlevel10k#meslo-nerd-font-patched-for-powerlevel10k>

字体设置

Powerlevel10k 主题支持多种字体，不过最好适配的是 Meslo Nerd Font 字体。可在主题链接中找到字体下载链接。下载安装字体后，可在使用的终端工具中启用该字体

Vscode配置：

Open File → Preferences → Settings (PC) or Code → Preferences → Settings (Mac), enter terminal.integrated.fontFamily in the search box at the top of Settings tab and set the value below to MesloLGS NF. Consult this screenshot to see how it should look like or see this issue for extra information.

下载安装

oh-my-zsh 安装该主题使用以下命令即可

git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

下载安装完成后，在 ~/.zshrc 文件中修改 ZSH_THEME 项为 ZSH_THEME="powerlevel10k/powerlevel10k"。然后使用命令 source ~/.zshrc 使配置生效

主题配置

在主题下载安装好后的第一次使用中，会让用户配置各种项目，按照个人观感进行选择。如果需要重新设置，使用命令 p10k configure 即可
